National Cyber Threat Intelligence Hub

Overview
The National Cyber Threat Intelligence Hub is a real-time cyber threat monitoring and analysis platform. It collects, processes, and visualizes attack data from multiple sources—including honeypots and external threat intelligence APIs—to produce actionable insights for security teams, researchers, and analysts.

Tech Stack

Frontend: React

Backend: Node.js, Express

Database: MongoDB

Threat Data Source: Cowrie Honeypot + External APIs

Deployment: Docker
